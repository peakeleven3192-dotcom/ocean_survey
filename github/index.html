本調査は◯◯湾沖における水温・塩分・溶存酸素・クロロフィルaの鉛直分布を把握することを目的とし、2024年6月から8月にかけて実施しました。調査船◯◯丸を用い、計24測点においてCTD観測および採水を行いました。

05

解析結果・考察

2024年夏季の調査海域における表層水温は23.6〜25.3℃の範囲にあり、前年同期と比較して平均0.4℃高い傾向が認められました。塩分は外洋側ほど高く、湾奥では低塩分水の影響が確認されました。

クロロフィルa濃度は沿岸域（St.01〜03）で高く、外洋側で低い傾向を示しました。鉛直分布では25〜40m層に極大層が形成されており、光環境と栄養塩供給のバランスを反映していると考えられます。
